Web Basics Cheat Sheet

World Wide Web

Invented by Tim Berners-Lee in 1989, the WWW uses three core technologies: HTML (markup language), URL (addressing system), and HTTP (communication protocol).

URL Syntax

A URL includes a scheme (e.g., https), host (e.g., www.example.com), and path (e.g., /index.html). It may also include query parameters and fragments.

Web Server

A web server stores, processes, and delivers web pages to clients using HTTP. It handles requests from browsers and serves HTML files or other resources.

Web Browser

A browser is a software application that retrieves and displays content from the web. It interprets HTML, CSS, and JavaScript to render pages.

<!DOCTYPE>

Declares the document type and version of HTML. Example: <!DOCTYPE html> tells the browser to use HTML5.

<html>

The root element of an HTML document. All other elements are nested inside it.

<head>

Contains metadata, links to stylesheets, and the document title. It does not display content on the page.

<title>

Sets the title of the web page, shown in the browser tab.

<meta>

Provides metadata like character encoding. Example: <meta charset="utf-8">

<body>

Contains the visible content of the web page, such as text, images, and links.

<em>

Emphasizes text, usually displayed in italics. Example: <em>important</em>

<strong>

Indicates strong importance, usually displayed in bold. Example: <strong>urgent</strong>
